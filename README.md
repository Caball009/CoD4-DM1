# CoD4 DM1
Reverse CoD4 & CoD4X ``.DM_1`` demo files.

## Working
* Protocol 16/17/18
* Read snapshots and frames
* Parsing gamestate
* Parsing snapshot
* Parsing entities
* Parsing clients
* Parsing server messages
* CoD4 & Q3 huffman code
* Msg parsing
